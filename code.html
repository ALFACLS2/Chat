Apa itu Gemini?

10:42 AM
auto_awesome Gemini Intelligence

Gemini adalah model bahasa besar (LLM) terbaru dan paling mumpuni dari Google. Ia dirancang untuk menjadi multimoda sejak awal, yang berarti ia dapat memahami, mengoperasikan, dan menggabungkan berbagai jenis informasi secara mulus.

  • Multimodalitas Native: Memproses teks, gambar, video, audio, dan kode secara bersamaan.
  • Tiga Ukuran: Dioptimalkan untuk Ultra (tugas kompleks), Pro (skalabilitas), dan Nano (perangkat mobile).

Apakah Anda ingin melihat contoh kode bagaimana cara mengintegrasikannya?

/api/chat.js (Vercel Function) content_copy 
import { GoogleGenerativeAI } from "@google/generative-ai";

export default async function handler(req, res) {
  const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
  const model = genAI.getGenerativeModel({ model: "gemini-pro" });

  const prompt = req.body.prompt;
  const result = await model.generateContent(prompt);
  const response = await result.response;
  
  res.status(200).json({ 
    text: response.text() 
  });
}
